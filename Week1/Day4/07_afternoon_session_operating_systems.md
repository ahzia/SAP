### Think About Your Computer

**Activity:** Think about your computer or phone. What happens when you turn it on?

**What you see:**

- Icons and apps

**What you don't see (but is working):**
- System managing everything

- Files being organized

**This is the operating system working!**

---

### What is an Operating System?

**Definition:** An operating system (OS) is software that manages computer hardware and software resources, and provides services for computer programs.

**In Simple Terms:** The operating system is like a manager that:
- Controls the computer hardware
- Manages all programs
- Organizes files and data
- Lets you interact with the computer

**Think of it as:** The "boss" of your computer that makes everything work together.

**Mobile Devices:**

## ðŸŽ¨ Part 3: OS vs. Platform - Understanding the Difference

### What is a Platform?

**Definition:** A platform is a computing environment where applications can run and develop.

**In Simple Terms:** A platform provides the foundation and tools for applications to work.

**Examples:**
- Windows Platform (runs Windows apps)
- iOS Platform (runs iPhone apps)
- Cloud Platform (runs cloud applications)
- SAP Platform (runs SAP applications)

---

### OS vs. Platform Comparison

**Operating System:**
- Manages hardware and software
- Controls computer resources
- Provides basic services
- Examples: Windows, Linux, macOS

**Platform:**
- Environment for applications
- Provides development tools
- Can include OS + additional tools
- Examples: Windows Platform, Cloud Platform, SAP Platform

**Key Difference:**
- **OS** = Manages the computer
- **Platform** = Environment for applications

### SAP and Operating Systems

**SAP systems run on operating systems!**

**Common Operating Systems for SAP:**
- **Windows Server** - For SAP applications
- **Linux** - Very popular for SAP (especially SAP HANA)
- **Unix** - For large SAP systems
- **Cloud Platforms** - SAP Cloud runs on cloud OS

**Why does SAP need an OS?**
- To manage system resources
- To run SAP applications
- To handle data storage
- To manage network connections

**SAP Kernel:**
- Manages SAP system resources
- Controls memory and CPU
- Handles SAP processes

- **What it provides:** Tools, services, and environment for development

- Accessible from anywhere
- No need to manage servers
